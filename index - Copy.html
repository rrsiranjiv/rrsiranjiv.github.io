About me

Hi, I'm Siranjiv. A Passionate Web Developer and Data Science Enthusiast I build highly scalable and responsive website using technologies such as React and Flux architecture (Redux). I do have professional experience in creating Single Page Application, Landing Pages and Chat Dashboard.

I have developed team collaboration tools, inhouse fun gaming product and real time real time audio/video communication application using on-demand technologies such as Websockets and WebRTC. Also I have created starter kits for web projects, to kick start development right away without writing boilerplate code and going through the whole setup process.

I have also developed scalable web servers using nodejs, RESTful services using express.js framework, chat application using socket.io, worked with viewengine such as handlebars and tested them using mocha, expect and supertest

I do have some experience in creating chatbots using python, using third party services such as API.ai, rasaNLU and Chatterbot.

Augury is an inhouse fun game product created specifically for Indian Premier League. The product is basically about predicting the winning team to gain points in order to progress in your leaderboard.

Augury can be described as a good example of Rapid Software Development(RAD), because it's more of a adapative approach rather than linear sequential design approach where the progress flows largely in one direction. It was completely designed, developed and deployed all by developers with a span of 2 weeks, which we considered as a remarkable achievement.

It was built using ReactJS, Redux for front end developement and golang for server side developement. MySQL is used as data storage layer.
